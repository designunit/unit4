## АЙДЕНТИКА ЗАГОРОДНОГО ПАРКА

<Image src='https://s.tmshv.com/unit4/samarapark-07.jpg'/>

Aдентика Загородного парка строится на следующих принципах:
- природность *(концентрические годичные кольца дерева, естественный рельеф, характерный для парка)*
- антропогенность, влияние и участие человека *(символическое изображение отпечатка пальца)*
- самообновление, взаимодействие, изменение, обменный процесс *(проекция принципа сложного
  взаимодействия таких компонентов, как: Человек, Ландшафт, Время)*

## ПРОЕКТНЫЕ РЕШЕНИЯ : : ФУНКЦИОНАЛ

<Image src='https://s.tmshv.com/unit4/samarapark-02.jpg'/>
<Image src='https://s.tmshv.com/unit4/samarapark-03.jpg'/>

Существующий функционал общественных сервисов парка дополняется новыми категориями:

- по характеру размещения функций *(постоянные, временные, стихийные)*
- по сценариям времяпрепровождения - *(быстрые, медленные)*
- по степени "натуральности" - *(природные, антропогенные)*
- по качеству самореализации - *(участник-потребитель, участник-созидатель)*
- по градиенту активности - *(тихие, громкие)*

С опорой на существующую планировочную структуру пространство дополняется несколькими тематическими сетками - маршрутами:
- быстрые *(фуникулер, велодорожка, лыжная тропа)*
- спокойные *(регулярные пешеходные маршруты)*
- медленные *(нерегулярные лесные тропы)*
- регулярные *(инфраструктурная ось)*
- хаотичные *(сменяемая тропиночная сеть)*

**ВАЖНО!** Сохранить аутентичные функции, придав им новое качество дизайна и пространственного окружения: конный клуб, веревочный парк, катальная горка, переосмысление колеса обозрения, тропиночной сети.


## ПРОЕКТНЫЕ РЕШЕНИЯ : : МОДЕЛЬ ФУНКЦИОНИРОВАНИЯ

<Image src='https://s.tmshv.com/unit4/samarapark-05.jpg'/>

Формула функционирования площадок парка:
- В планировочной структуре парка содержатся несколько площадок с функциями на долгосрочной
  контрактной основе. В этом ряду есть несколько вакантных площадок на будущее развитие.
  Правило в том, что основа контракта - всесезонное использование *(или использование на
  срок не менее одного сезона, регулярный контракт),* и две площадки должны быть всегда свободны
  для будущих функций. Они могут быть заняты, если высвобождаются другие площадки по истечении контракта.
- Такая структура обеспечивает сменяемость операторов схожих функций, либо располагает их
  к смене оборудования при возобновлении контракта.
- Cистема располагает к тому, чтобы множились инициативы, примеривающиеся к использованию
  площадок в парке, к возделыванию инициативами своей аудитории,
  к разнообразию аудиторий и способов использования площадок парка.

<Image src='https://s.tmshv.com/unit4/samarapark-06.jpg'/>

Зеленые площадки парка в незаполненные таймслоты "регенерируются" после интесивного использования.
